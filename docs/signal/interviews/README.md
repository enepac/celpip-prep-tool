# Async Written Interviews — Channel 4 Archive

One file per completed interview. Channel 4 starts day 14 (ramping from 1/week if recruitment lags, targeting 3/week).

## Filename format

```
YYYY-MM-DD-respondent-id.md
```

Use the date the response was received. Respondent IDs are anonymized sequentially: `r001`, `r002`, etc. No real names or email addresses in filenames or file content.

## File structure

```markdown
# Interview — YYYY-MM-DD — respondent-id

**Respondent country:** [country]
**CELPIP history:** [never taken / taken once (score) / taken N times / not shared]
**Incentive paid:** yes / no
**Date sent:** YYYY-MM-DD
**Date received:** YYYY-MM-DD

## Q&A

**Q1:** [question text]
> [verbatim or close-paraphrase response]

**Q2:** [question text]
> [verbatim or close-paraphrase response]

[... repeat for all questions]

## Notes

[Any context worth noting: tone, follow-up threads, signals not captured in the Q&A]
```

## Recruitment and question framing

Questions follow Mom Test principles: ask about past behavior and current problems, not hypothetical future use. Avoid leading questions that invite validation. Example framing: "Walk me through the last time you practiced CELPIP Speaking. What did you do?" not "Would you use a tool that gave AI feedback?"

Incentive: $20 e-gift card or 3 months free Premium per completed response. State the incentive clearly in the recruitment message.

## Cull check

Response rate below 20% across 15 invitations → framing or audience is wrong. Restructure the question set or target a different recruitment channel before continuing.
